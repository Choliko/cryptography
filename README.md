Fast SHA-256 implementation for JavaScript

## Setup
```
npm install @cryptography/sha256
```
```
yarn add @cryptography/sha256
```

## Usage
```js
import sha256 from '@cryptography/sha256'

sha256('Hello World!') // returns string
```
